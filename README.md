# 🌿 HayFarm Admin Dashboard

> Sistem manajemen peternakan modern untuk mengelola produk, kesehatan hewan, dan verifikasi penjualan dalam satu dashboard elegan.

---

## 🚀 Fitur Utama

### 🧭 Dashboard

* Ringkasan statistik cepat
* Tampilan bersih & modern
* Navigasi sidebar interaktif

### 📦 Manajemen Produk

* CRUD produk
* Kategori produk (Hewan, Susu, Rumput)
* Status stok & ketersediaan

### ✅ Verifikasi Penjualan

* Validasi pesanan
* Status: Menunggu, Diverifikasi, Ditolak
* Filter & pagination

### 🐄 Data Hewan

* Informasi lengkap ternak
* Status produktivitas
* Tracking kandang & berat

### ❤️ Data Kesehatan Hewan

* Monitoring kesehatan real-time
* Status: Sehat, Perawatan, Observasi
* Riwayat diagnosa & tindakan

---

## 🎨 UI/UX Design

* 💚 Warna dominan hijau natural (agrikultur feel)
* 📱 Responsive layout

## 🔗 Navigasi Halaman

Semua halaman sudah terhubung melalui sidebar:

* Dashboard → `index.html`
* Produk → `produk.html`
* Verifikasi → `verifikasi.html`
* Data Hewan → `data-hewan.html`
* Kesehatan → `kesehatan.html`
